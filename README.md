## Get started

Install the dependencies...

```bash
cd master-yey
cd client
yarn install
cd ..
cd server
yarn install
```

...then start the client and server:

```bash
cd client
yarn dev
```

```bash
cd server
yarn dev
```

Navigate to [localhost:5173](http://localhost:5173). You should see the game application running.
